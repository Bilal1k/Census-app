This shiny web app generates interactive chloropleth maps using data from Statistics Canada's 2016 census. 
At the moment, it has visible minority data in the GTA. In the Future mutiple metropolitan areas and other variable will be added.
It was developed using R version 4.0.5 and RStudio verrsion 1.4.1106.
For a deployed version, check

https://01jrh3-bilal1k.shinyapps.io/census/
