This shiny app generates choropleth interactive maps of demographic data for all major census metropolitan areas in Canada (Population over 900,000) based on the 2016 census at the census tract (CT) level.

It was developed using R version 4.0.5 and RStudio verrsion 1.4.1106.

For a deployed version, check
https://01jrh3-bilal1k.shinyapps.io/census/
